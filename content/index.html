Easy dashboards and data storytelling with MelViz!

A fast, easy-to-use, cloud-native and open source data visualization tool for software developers.

Melviz is a Dashbuilder fork!

Create dashboards your way

MelViz supports dashboards created from YAML. You can make use of the dozen of available internal visual components or build your own using any Javascript library.

YML

Author dashboards in a browser client-side editor using YML language. Export a client-only application to use anywhere or keep it stored in the cloud. See [MelViz YAML Guide](/docs/melviz-yaml-guide)

Visual Components

MelViz comes with a full set of traditional visual components, but developers can extend the components library by building their own.

Connect to any source of data

Datasets can be in JSON format and be transformed using expression languages. Melviz natively also support metrics and CSV formats

Easy Deploy

MelViz is a full client side application which only requires a web server to be deployed. It can be easily deployed in a cloud native environment and connect to existing services and APIs.
